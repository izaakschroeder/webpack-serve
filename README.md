# webpack-serve

Something better than webpack-dev-server.

Features:
 * Isomorphic support.
 * Build out for development and production.
 * Hot module reload.
 * Not monolithic.
 * Serve static assets.
 * Server web routes.
 * Multiple webpack config files.

## Usage

```sh
npm install webpack-serve
```



```javascript
var app = express();
```
